Finding a Collatz Sequence

This snippet is a function that returns the complete Collatz Sequence for a given number (num). 

def collatz(num, array = [num])
  if num == 1
    array
  else
    if num % 2 == 0
      num = num / 2
    else
      num = 3 * num + 1
    end
  array << num
  collatz(num, array)
  end
end

Palindrome Check

This snippet is a function that determines whether a string is a palindrome or not. 

def palindrome?( some_string )
  _palindrome?( some_string, 0, some_string.length - 1 )
end

def _palindrome?( some_string, head_index, tail_index )
  if head_index == tail_index || head_index + 1 == tail_index && some_string[head_index] == some_string[tail_index]
    true 
  else
    if some_string[head_index] == some_string[tail_index]
      _palindrome?(some_string, head_index + 1, tail_index - 1)
    else
      false
    end
  end 
end

Instantiating a Class Object

This snippet is a class that instantiates a new object of a person's first and last name. 

class Person
  attr_accessor :first_name, :last_name

  def initialize(first_name, last_name)
    @first_name = first_name
    @last_name = last_name
  end

  def full_name
    "#{@first_name} #{@last_name}"
  end
end
